Dashboard de Vendas – Assinaturas Xbox (Bootcamp DIO)

Este projeto foi desenvolvido como parte do desafio "Criar um Dashboard de Vendas no Excel" do bootcamp da DIO. O objetivo é transformar dados brutos em informações visuais claras, organizadas e úteis, permitindo uma análise eficiente do desempenho e auxiliando na tomada de decisões baseada em dados.

O dashboard criado analisa ganhos referentes a assinaturas do ecossistema Xbox, como EA Play, Minecraft Season Pass e outros serviços. Ele apresenta visualizações de ganhos mensais, anuais e quadrimestrais, organizadas de forma a facilitar a interpretação dos dados e a identificação de padrões ao longo do tempo.

O arquivo principal do projeto é o documento Projeto_DIO_03.xlsx, que contém o dashboard finalizado, com tabelas estruturadas, gráficos, segmentações e indicadores.

Ferramentas utilizadas:
Microsoft Excel, com uso de tabelas dinâmicas, gráficos, segmentações de dados e formatações apropriadas para construção do dashboard.

Como utilizar:
Baixe o arquivo Projeto_DIO_03.xlsx, abra no Excel e explore o painel de indicadores e gráficos disponíveis. Caso deseje fazer adaptações, utilize as abas e tabelas como base para personalização.

Este repositório atende às exigências do desafio da DIO: criação do dashboard, organização do repositório, inclusão de README explicativo e disponibilização do arquivo final.
